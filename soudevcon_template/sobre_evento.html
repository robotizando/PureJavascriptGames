Sobre o Evento

A SouDevCon é uma conferência dedicada à celebração das linguagens de programação e ao desenvolvimento de software. O evento reúne profissionais, estudantes e entusiastas para compartilhar conhecimentos, experiências e melhores práticas no uso das principais linguagens do mercado.

Na edição de 2025, o foco principal está nas linguagens que moldam o futuro do desenvolvimento web e de aplicações:

Python - Versatilidade e simplicidade para desenvolvimento rápido
JavaScript - Interatividade e dinamismo para web e além
PHP - Robustez e confiabilidade para aplicações web
